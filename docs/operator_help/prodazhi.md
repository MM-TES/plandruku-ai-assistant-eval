# Вкладка «Продажі»

Тут зібрані нові замовлення, для яких ще **не обрано матеріал** — планувальник їх ще
не опрацював. Завдання вкладки: затвердити, який матеріал піде на кожен компонент замовлення.

## Що ви бачите
Список замовлень із клієнтом, структурою (матеріалом) і терміном. Біля кожного — підказка
системи, який матеріал імовірно підходить (на основі історії). Ця підказка — лише
орієнтир: остаточний вибір матеріалу ви робите з довідника аналогів.

## Що робити
1. Натисніть **Опрацювати** біля замовлення.
2. Відкриється екран **вибору матеріалу**. На кожен компонент оберіть матеріал з
   **довідника аналогів**: вид матеріалу, **стандарт-код** (марку), за потреби —
   виробника/конкретну марку; товщина підставляється із замовлення.
3. Натисніть **🔎 Показати рулони**, щоб одразу побачити, що є на складі під ваш вибір.
4. Збережіть. Після цього замовлення перейде на вкладку **Матеріали**, де підбирають
   конкретні рулони.

Підказка: починайте з замовлень із найближчим терміном. Обираючи **стандарт-код**, ви
автоматично охоплюєте всі валідовані аналоги всіх виробників — це найшвидше.
